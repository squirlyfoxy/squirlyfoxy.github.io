Commands

  • help - Prints the list of commands
  • exit - Exits from Ronny
  • wami - Show the name of the database
  • chme [newName] - Change the name of the database
  • tlist - show all the tables
  • shtable [tableID] - show a table structure
  • rash - clean all scripts hashes
  • save - save the database
  • serve - start the API web server
  • realod scripts - reload the database resetting the data structures
