Example of a Ron Function

Ron transpiles functions to Golang, every function is an executable program.

Every function refers to the table where it is defined (called "scope"). In a scope we can loop through all the rows as you can see in the example below.

Example

                        
@function (TakeUsersWithAgeGreaterThan18)
{
    @var (Users) _usrs;
                            
    loop on * as user
    {
        if user.Date_Of_Birth.CalculateAge() >= 18
        {
            _usrs.Add(user);
        }
    }
                                
    return _usrs;
}
